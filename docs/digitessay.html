DIGIT 100 Essay

Overall I enjoyed working with both tools to build websites, and feel that I learned a lot of information about

each that I didn’t know before I took this class. It’s hard for me to decide which tool I liked better, I feel

WordPress was a lot easier to work with for me but I also recognize that GitHub seems to have a lot more

potential and let you do so much more once you understand it. Initially I struggled with understanding how

GitHub/GitBash/Oxygen worked, but overtime I feel I’ve become more proficient at using these tools and

now see how useful it is. I think that once I gain a full understanding of GitHub, I will like it significantly

more than WordPress, but for just starting out WordPress was a little easier to use. I did however run into a

lot of small problems with WordPress and also felt very limited in what I could do. While it was nice having

templates or preset components to the website, I wasn’t able to customize every small detail which is a reason

why I think I would like GitHub better in the future. One thing about GitHub that I like a lot more than

WordPress was how well it kept track of your progress and would allow you to revert to previous points in

the project. That was extremely helpful and something that I felt was lacking in WordPress. All in all I believe

that GitHub is the better tool to use, but for a beginner WordPress was a little easier to use.
